Instrucciones para el Equipo Externo

URLs de WebSocket:

  • Canal de visitantes: ws://tu-dominio:8080/app/REVERB_APP_KEY
  • Canal de estadísticas: ws://tu-dominio:8080/app/REVERB_APP_KEY

Eventos Disponibles:

  • visitante.actualizado - Cuando un visitante entra o sale
  • estadisticas.actualizadas - Cuando se actualizan las estadísticas

Endpoints HTTP:

  • GET /websocket/estadisticas - Obtener estadísticas actuales
  • POST /websocket/entrada - Registrar entrada de visitante
  • POST /websocket/salida - Registrar salida de visitante
  • GET /websocket/visitantes-actuales - Obtener lista de visitantes actuales
